Time and Space Complexity

The valid algorithm takes a finite amount of time for execution and efficent. The Time Complexity required by the algorithm to solve given problem is called time complexity of the algorithm. Time complexity is very useful measure in algorithm analysis. It is the time needed for the completion of an algorithm. To estimate the time complexity, we need to consider the cost of each fundamental instruction and the number of times the instruction is executed.

Problem-solving using computer requires memory to hold temporary data or final result while the program is in execution Space Complexity. The amount of memory required by the algorithm to solve given problem is called O(n) Space complexity of the algorithm.
